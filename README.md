## Projects

### Energy Demand Forecasting
LSTM-based hourly energy demand prediction with a real-time FastAPI dashboard and React frontend.

- **Stack:** TensorFlow 2.19 · FastAPI · React · Pandas · SQLite
- **Data:** UCI Household Power Consumption dataset
- **Highlights:** Multi-layer LSTM, EarlyStopping, REST API, live chart dashboard

### Predictive Maintenance for Manufacturing
XGBoost ensemble that predicts machine failures before they happen on a synthetic 18,250-record sensor dataset.

- **Stack:** XGBoost · scikit-learn · SMOTE · Optuna · Power BI
- **Data:** AI4I 2020 Predictive Maintenance Dataset
- **Results:** 95% accuracy · AUC 0.85 · F1 0.75 · Precision 0.82 · Recall 0.69

[View Project](Predictive_Maintenance_Manufacturing/)

---

### Zero-Shot Visual Knowledge Base — MSc Dissertation
Scene classification system that requires zero labelled test-time data, using YOLOv8 object detection and a TF-IDF Visual Knowledge Base matched against Places365 scene labels.

- **Stack:** YOLOv8 · Python · TF-IDF · Places365
- **Results:** 68.5% baseline → 77% valid-pred accuracy · 80.25% Top-2 · 10.75% abstention rate
- **Significance:** Eliminates dependency on labelled test data entirely

[View Project](Zero-Shot%20VKB/)

---

### VoiceChef
Voice-activated recipe assistant — speak your available ingredients and get a recipe with carbon footprint analysis and full nutritional breakdown.

- **Stack:** OpenAI Whisper · FastAPI · React · Spoonacular API
- **Highlights:** Speech-to-text pipeline, environmental impact scoring, ingredient-based matching

[View Project](voice-to-recipe/)
