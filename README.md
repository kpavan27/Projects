## Projects

### Energy Demand Forecasting
LSTM-based hourly energy demand prediction with a real-time FastAPI dashboard and React frontend.
- **Stack:** TensorFlow 2.19 · FastAPI · React · Pandas · SQLite
- - **Data:** UCI Household Power Consumption dataset
  - - **Highlights:** Multi-layer LSTM, EarlyStopping, REST API, live chart dashboard
    - - [View Project](Energy_Demand_Forecasting/)

      ### Predictive Maintenance for Manufacturing
      XGBoost ensemble that predicts machine failures before they happen on a synthetic 18,250-record sensor dataset.
      - **Stack:** XGBoost · scikit-learn · SMOTE · Optuna · Power BI
      - - **Results:** 95% accuracy · AUC 0.85 · F1 0.75 · Precision 0.82
        - - **Highlights:** 53→30 feature selection, Voting Classifier, F1-threshold optimisation, Power BI dashboard
          - - [View Project](Predictive_Maintenance_Manufacturing/)
           
            - ---

            ### Zero-Shot Scene Classification — MSc Dissertation
            Interpretable computer vision pipeline that classifies real-world scenes without any labelled test data, using YOLOv8 and a custom Visual Knowledge Base (VKB).
            - **Institution:** Technological University Dublin · M.Sc. Computer Science (Data Science) · Sep 2025
            - - **Stack:** YOLOv8-Large · Python · TF-IDF · Places365 dataset
              - - **Results:** 77% valid-prediction accuracy · 80.25% Top-2 containment · 10.75% abstention rate
                - - **Highlights:** TF-IDF weighted object profiles, principled abstention rule, fully explainable outputs
                  - - [View Project](Zero-Shot%20VKB/)
                   
                    - ---

                    ### VoiceChef — Voice-to-Recipe Generator
                    AI application that converts voice recordings of available ingredients into full recipes with carbon footprint analysis and nutritional breakdown.
                    - **Stack:** OpenAI Whisper · FastAPI · React · Browser MediaRecorder API
                    - - **Highlights:** Speech-to-text transcription, ingredient extraction, sustainability scoring, responsive mobile UI
                      - - [View Project](voice-to-recipe/)
