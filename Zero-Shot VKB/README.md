# Zero-Shot VKB

This project is a Zero-Shot VKB experiment. 

## Usage

To run this project, you will need to have Jupyter Notebook installed. You can then open the `final (3).ipynb` notebook and run the cells.

## Data

The data for this project is not included in this repository. You will need to download it separately and place it in the `drive-download-20250922T101253Z-1-001/` directory.
